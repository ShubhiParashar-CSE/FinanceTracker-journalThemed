# Finance Tracker (Journal Themed)

A simple web app to help you track your personal finances with a cozy journal-like interface, making budgeting feel less like a chore and more like jotting down notes.

## Tech Stack
- HTML, CSS, JavaScript
- Google Fonts (Playfair Display and Nunito for that handwritten feel)

## Features
- Track your income and expenses with easy categorization
- Set up budget envelopes by allocating percentages of your income to different categories
- Monitor spending limits per category with visual progress bars
- View all transactions in a notepad-style log with filter tabs

## How to Run
Just open the `index.html` file in any modern web browser. No server or installation needed!

## What I Learned
Building this project taught me a lot about how the envelope budgeting system works in practice — it's way more intuitive than I thought. I also got better at DOM manipulation in vanilla JavaScript and creating responsive layouts with CSS Grid. Plus, styling with custom properties made the color theming super flexible!
